## 📖 About The Project

This repository contains the **official website for InAmigos Foundation**, built end-to-end as the primary deliverable for my Web Developer Internship. The goal was simple but demanding: design and ship a clean, professional, fully responsive site that represents a real organization — no templates, no shortcuts.

The site was built from a blank canvas: structure, styling, imagery, and responsiveness were all handled independently, with a focus on making it look like something a real NGO/foundation would confidently put in front of visitors and donors.

> 💡 **Why this matters:** This wasn't a tutorial clone or a classroom assignment — it's a live deliverable, built under a real timeline, for a real organization, now deployed and publicly accessible.

<br>

## ✨ Key Features

| Feature | Description |
|---|---|
| 🎨 **Custom Design** | Fully hand-coded layout — no page builders, no templates |

Clone the repo and open it directly — no build steps, no dependencies to install.

```bash

# Navigate into the project
```
